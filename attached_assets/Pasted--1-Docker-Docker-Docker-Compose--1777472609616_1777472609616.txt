Шаг 1 — Установка Docker на сервере
Устанавливаем Docker и Docker Compose. Это разовая операция.

Шаг 2 — Разворачиваем Excalidraw с коллаборацией
Клонируем alswl/excalidraw-collaboration на сервер, настраиваем docker-compose.yml — указываем домен kvant-academy.ru, порты. Запускаем контейнеры. Excalidraw будет работать на каком-то локальном порту, например 3002.

Шаг 3 — Настраиваем nginx
Добавляем в существующий nginx конфиг два новых location:
nginxlocation /board/ {
    proxy_pass http://127.0.0.1:3002/;
    proxy_http_version 1.1;
    proxy_set_header Upgrade $http_upgrade;
    proxy_set_header Connection "upgrade";
}
WebSocket заголовки обязательны — без них совместная работа не работает.

Шаг 4 — Схема БД
Добавляем поле board_url в таблицу bookings. При создании записи на занятие автоматически генерируется уникальный roomID (UUID) и сохраняется в этом поле.

Шаг 5 — Логика в боте
Когда пользователь записывается на занятие — бот генерирует roomID и сохраняет ссылку kvant-academy.ru/board/[roomID] в БД. За 30 минут до занятия планировщик (уже есть в боте) отправляет ученику ссылку в Telegram.

Шаг 6 — Личный кабинет на сайте
В ЛК ученика добавляем раздел "Мои занятия" — список занятий с датами и кнопкой "Открыть доску" которая ведёт на kvant-academy.ru/board/[roomID]. Доска открывается прямо в браузере.

Шаг 7 — Страница доски на сайте
Добавляем роут /board/:roomId в kvant-site. На этой странице просто iframe или встроенный Excalidraw который загружает нужную комнату. Выглядит как часть сайта.

Нюансы которые важно учесть:

Excalidraw в Docker потребляет ~200-300MB RAM — у тебя 4GB должно хватить
WebSocket соединения — nginx должен их поддерживать, у тебя nginx 1.28 это умеет
Доски хранятся в контейнере — нужно настроить volume чтобы данные не терялись при перезапуске
SSL уже есть на домене — Excalidraw через nginx будет работать по HTTPS автоматически
Если Docker контейнер упадёт — доски недоступны, бот продолжает работать. Нужен мониторинг.
Старые доски занятий хранятся вечно — нужно решить когда их удалять (например через 30 дней после занятия)
